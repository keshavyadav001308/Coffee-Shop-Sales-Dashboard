Coffee Shop Sales Analysis & Dashboard

 Project Overview
An end-to-end **data analytics project** that analyzes coffee shop sales data using **MySQL, Python (Jupyter Notebook), and Power BI** to generate actionable business insights.

 Tech Stack
- **Python (Jupyter Notebook)** – Data cleaning & preprocessing  
- **MySQL** – Querying and business problem solving  
- **Power BI** – Interactive dashboard & visualization  

 Key Analysis Performed
- Sales, revenue, and transaction analysis  
- Product category & store location performance  
- Monthly, daily, and hourly sales trends  
- KPI calculation using SQL & DAX  

Dashboard Highlights
- Total Revenue, Quantity Sold & Avg Transaction Value  
- Location-wise & category-wise performance comparison  
- Time-based sales insights with interactive filters  

 Project Workflow
1. Data cleaning & preprocessing in **Jupyter Notebook**  
2. Business queries solved using **MySQL**  
3. Data modeling & visualization in **Power BI**
